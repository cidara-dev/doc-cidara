##### Template ini bertujuan memudahkan pengerjaan awal sebuah web,tanpa manual install tailwind.
##### Selanjutnya kita tinggal masukkan dependensi lain kedalam struktur web yg akan kita buat.

## Pre Install

- ***nodejs*** + ***npm***
- ***visual studio code***

## Installation

```sh
git clone https://github.com/sdldev/tailwindcss-layout.git
```

```sh
cd tailwindcss-layout
```

```sh
npm install
```

```sh
npm run dev
```

- Buka file html  di ``` src/``` menggunakan visual studo code, live server
- edit html file
