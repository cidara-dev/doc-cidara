###### Untuk Menginstall web server di Windows,bisa menggunakan *XAMPP* ,yang didalamnya sudah terdapat *Apache*,*MariaDB*,*PHP* dan *Perl*.Anda dapat mengunduhnya di 
- [XAMPP Latest Version](https://sourceforge.net/projects/xampp/files/latest/download)
- [XAMPP-8.2.12/PHP 8.2.12](https://sourceforge.net/projects/xampp/files/XAMPP%20Windows/8.2.12/xampp-windows-x64-8.2.12-0-VS16-installer.exe/download) 
- [XAMPP-8.1.25/PHP 8.1.25](https://sourceforge.net/projects/xampp/files/XAMPP%20Windows/8.1.25/xampp-windows-x64-8.1.25-0-VS16-installer.exe/download)

Untuk Versi lebih lengkap bisa kunjungi Link [XAMPP](https://sourceforge.net/projects/xampp/files/XAMPP%20Windows/)
